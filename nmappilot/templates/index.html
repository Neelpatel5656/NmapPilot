NmapPilot uses OpenRouter to access free AI models. All models are free — no charges.

🔑 OpenRouter API Key

To get a free API key:

  1. Go to openrouter.ai
  2. Sign up (free — no credit card needed)
  3. Go to Keys → Create Key
  4. Copy the key and paste it below
🧠 AI Model

Select preferred model. If it fails, NmapPilot auto-switches to the next free model.

🏠 Local Ollama (Fallback)

If all OpenRouter models fail, NmapPilot will try your local Ollama instance. Make sure Ollama is running:

ollama serve
